# MTC-to-DenD

MTCの入力をキーボード入力に変換します。<br>
電車でd shiningstageでの利用を想定しています。
<br><br>
ライブラリとしてLibUsbDotNet( https://sourceforge.net/projects/libusbdotnet/ )を活用しました。<br>
LibUsbDotNet及びドライバはLGPL v3が適応されます。
<br><br>
このソフトはMTC P4-B7+非常 専用です。他のカセットでは動作しません。
<br><br>
キーコンフィグ<br>
力行P1-4段 : "A"<br>
N段 : NULL<br>
ブレーキB1-7段 : "S"<br>
ブレーキ非常段 : "D"<br>
やじる上下左右 : 上下左右<br>
ボタンA : "Q"<br>
ボタンA深押し : NULL<br>
ボタンB : "エンター"<br>
ボタンC : NULL<br>
ボタンD : "W"<br>
ボタンATS確認 : NULL<br>
ボタンSELECT : NULL<br>
ボタンstart : "Esc"<br>
